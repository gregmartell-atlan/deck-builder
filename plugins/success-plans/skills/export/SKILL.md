# Success Plan Export

You export a Success Plan (generated by `/success-plans:plan`) to a Google Sheet and a Google Slides deck.

## Prerequisites

- A Success Plan must exist in the current conversation (generated by `/success-plans:plan`)
- If no plan exists, tell the user: "Run `/success-plans:plan {customer}` first to generate a plan, then run `/success-plans:export` to export it."

## Parameter Collection

Parse `$ARGUMENTS`:
- **Customer name** (optional): confirms match with the plan in conversation. If omitted, use the customer from the most recent plan.

## Context Retrieval

From the current conversation, extract:
- The final refined OKR table (all 14 columns)
- The Key Results list (Key Result 1, 2, 3... with full descriptive text)
- Strategic Objectives and Domains table
- Enabling Initiatives list
- Customer name, scenario type, year
- Any Snowflake usage data that was retrieved (MAU, adoption, retention)

### Safe Text Styling

**ALWAYS use `styled_element()` for multi-styled text.** Never hardcode character indices.

```python
def styled_element(oid, segments):
    """Build text from segments, auto-computing indices.
    Each segment: (text, sz, bold, color). Returns ops list."""
    full = ''.join(s[0] for s in segments)
    ops = [
        {'deleteText': {'objectId': oid, 'textRange': {'type': 'ALL'}}},
        {'insertText': {'objectId': oid, 'insertionIndex': 0, 'text': full}},
    ]
    idx = 0
    for text, sz, bold, color in segments:
        end = idx + len(text)
        s = {'fontFamily': FONT}
        fields = ['fontFamily']
        if sz:
            s['fontSize'] = {'magnitude': sz, 'unit': 'PT'}
            fields.append('fontSize')
        if bold is not None:
            s['bold'] = bold
            fields.append('bold')
        if color:
            s['foregroundColor'] = {'opaqueColor': {'rgbColor': color}}
            fields.append('foregroundColor')
        ops.append({'updateTextStyle': {
            'objectId': oid,
            'textRange': {'type': 'FIXED_RANGE', 'startIndex': idx, 'endIndex': end},
            'style': s, 'fields': ','.join(fields),
        }})
        idx = end
    return ops
```

## Graceful Degradation

| Failure | Behavior |
|---------|----------|
| No plan in conversation | Tell user to run `/success-plans:plan` first. Do not proceed. |
| Plan was Lite Mode (missing Enabling Initiatives or Strategic Objectives table) | For slide 2 Row 2, derive initiative text from KR descriptions. For slide 2 Row 1, use STRATEGIC OBJECTIVES column from OKR table. |
| Google auth token missing or expired | Tell user: "Run `/deck:ebr` or any prior deck script to authenticate Google, then re-run `/success-plans:export`." |
| Sheet script succeeds but deck script fails | Report the Sheet URL that was created. Show the deck error. Suggest re-running the deck script. |
| Deck script succeeds but sheet script fails | Report the Deck URL that was created. Show the sheet error. Suggest re-running the sheet script. |
| Missing data for slides (no usage metrics) | Use `[CUSTOMIZE]` placeholders in ADOPTION METRICS cells. Omit usage-specific content from slides. |
